Analyse actuarielle d’un portefeuille épargne-retraite

KPI • Qualité des données • Restitution métier — SAS / Excel

Contexte & objectif

Mettre en place une analyse structurée du portefeuille (profil assurés, contrats, encours), afin de produire des indicateurs fiables pour le pilotage et d’identifier des axes de segmentation pertinents.

Données

  • Contrats, assurés, historique des encours
  • Variables : âge, ancienneté, produit, encours, versements, etc.
  • Source : extraction SAS + consolidation Excel

Approche

  • Contrôles qualité : doublons, valeurs manquantes, cohérences
  • Exploration : distributions, tendances, segmentations simples
  • Production KPI : encours, taux d’évolution, structures par segment
  • Restitution : synthèse managériale et recommandations

Résultats

  • Tableaux de bord KPI pour suivi mensuel
  • Segmentation par profils (âge / encours / produit)
  • Identification de points d’attention sur qualité des données

Tableau de bord Assurance Vie – Pilotage du Portefeuille (Power BI)

Contexte & Objectif : Développement d’un dashboard interactif pour le pilotage d’un portefeuille d’assurance vie, permettant de centraliser les indicateurs clés et de faciliter la prise de décision stratégique.

  • Méthodes & Outils : Power BI pour la visualisation interactive. Structuration des données via Excel et modélisation analytique sous Python.
  • Résultats Clés & Impact : Suivi des encours, primes collectées, taux de rachat, répartition par type de support (UC / Fonds €), et analyse de l’évolution temporelle. Le dashboard permet un pilotage mensuel et une détection rapide des tendances.

Confidentialité : Le travail initial a été réalisé à partir de données réelles d’entreprise. Pour des raisons de confidentialité, une simulation de données cohérente a été générée afin de reproduire la structure analytique du projet tout en respectant les exigences de protection des données.

Outils : Power BI, Python, Excel

Impact décisionnel

  • Indicateurs fiables pour la prise de décision
  • Amélioration du pilotage du portefeuille
  • Base réutilisable pour reporting automatisé
